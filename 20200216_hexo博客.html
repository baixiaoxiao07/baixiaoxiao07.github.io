FAQ

hexo deploy错误

原因：需要github里设置ssh公钥

解决方法：查看github设置公钥

git命令没找到

问题：

原因：没有安装git插件

解决：命令行输入

_config.yml错误

原因：该文件所有参数后面接着的value需要加空格间隔开。。

中文乱码

问题：网站标题显示 “Hexo����”乱码

原因：配置文件_config.yml没有使用utf-8格式保存。

解决方法：配置文件_config.yml转换为utf-8格式保存。命令行输入

hero generate错误

原因：主要是因为某些插件升级后有bug，需要退版本

解决方法：

  • 删除node_modules

  • 把稳定版本的package.json和package-lock.json复制到当前文件夹并覆盖

  • npm install

    这个被坑了，重新安装后不识别hexo项目了

github域名解析

创建repository时候，注意是否重复了。因为访问{username}.github.io可以直接访问该博客。

例如我创建的hexo.github.io的域名别人被创建了，我一输入发现是别人的博客。囧。

注意 http://username.github.io中的username只能是github账号的username，不是随便取的

更换主题后空白

问题：

原因：因为_config.yml中提前设置了theme: next，将theme文件夹中的主题文件夹名字改为next即可
